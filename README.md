Dashboard Bio-Acuícola: Oncorhynchus Mykiss

Este proyecto es un Dashboard Interactivo de Análisis Acuícola enfocado en la optimización nutricional y el seguimiento del crecimiento de la Trucha Arcoíris (Oncorhynchus mykiss). El diseño utiliza una estética de Glassmorphism con efectos de iluminación neón para presentar datos técnicos de manera visualmente atractiva y moderna.

🚀 Características Principales

El dashboard presenta seis módulos críticos para la gestión de la producción:

Frecuencia vs. Ingesta Diaria: Gráfico de barras animado que muestra cómo la frecuencia de alimentación (v/día) impacta positivamente en la ingesta biológica.

Curva de Crecimiento: Gráfico vectorial (SVG) que compara el rendimiento de dietas de alta energía frente a dietas estándar mediante trayectorias animadas.

Eficiencia Alimenticia (FCR): Indicador de conversión alimenticia con un medidor de rango (Excelente, Óptimo, Promedio).

Estructura de Costos: Desglose porcentual de la inversión operativa (Alimento, Infraestructura, Otros) mediante barras de progreso dinámicas.

Proteína por Etapa: Comparativa de requerimientos proteicos entre etapas de Juveniles y Engorde.

Control de Temperatura: Termómetro dinámico que visualiza el rango térmico ideal para el crecimiento de la especie.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica de la aplicación.

CSS3 (Custom Properties & Modern UI):

Glassmorphism: Uso de backdrop-filter para efectos de transparencia y desenfoque.

Layout: Grid y Flexbox para una interfaz totalmente responsiva.

Animaciones: Transiciones suaves y efectos de hover con seguimiento de cursor (CSS variables dinámicas).

JavaScript (Vanilla):

Intersection Observer API: Las animaciones de los gráficos se activan solo cuando el usuario hace scroll hasta el elemento, optimizando el rendimiento.

Interacción de Tarjetas: Lógica para el efecto de brillo que sigue el movimiento del ratón en las tarjetas.

🎨 Diseño y Estética

El dashboard utiliza una paleta de colores neón sobre un fondo oscuro (#05070a) para maximizar el contraste:

Cian y Azul: Representan agua y condiciones óptimas.

Violeta y Rosa: Utilizados para resaltar datos de alto rendimiento o críticos.

Verde Neón: Indicador de éxito o eficiencia excelente.

📖 Fuentes de Referencia

Los datos y parámetros presentados están basados en estándares internacionales de acuicultura:

FAO (2023): Manuales técnicos de cultivo de salmónidos.

NRC: Requerimientos nutricionales de peces y crustáceos.

Alltech Coppens: Guías de alimentación industrial.
